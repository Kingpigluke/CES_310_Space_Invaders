Simple Space Invaders Game (Python + Arcade 3.x)

Project Description
This project is a simplified Space Invaders–style arcade game built using Python and the Arcade 3.x library.
The player controls a small ship that moves left and right, fires bullets, and attempts to destroy a moving enemy.
The game includes a background, an unmovable wall object, movable sprites, collision detection, and keyboard controls — all required features for the assignment.

Instructions for Build and Use
How to Build / Run the Software
Install Python 3.12 or later.

Install the Arcade library using:

Code
pip install arcade

Run the game file:

Code
python space_invaders.py

How to Use the Software
Use LEFT and RIGHT arrow keys to move the player.

Press SPACE to shoot bullets upward.

Hit the red enemy to destroy it.

Bullets disappear when they hit the wall or leave the screen.
